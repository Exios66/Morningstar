## How to Use Examples

### For Learning

1. **Read the markdown** — Understand the raw format
2. **Note the voice** — Sardonic but professional
3. **Follow the flow** — Opening → Arguments → Votes → Ruling
4. **Study the arguments** — Brief but substantive

### For Reference

When conducting your own deliberation:
1. Open a relevant example
2. Use it as a guide for structure
3. Adapt the format to your topic

### For Comparison

After conducting a deliberation:
1. Compare to examples
2. Check if you missed any elements
3. Ensure quality matches examples

## What Makes a Good Example

Examples in this directory should demonstrate:

1. **Proper Voice**
   - Sardonic, controlled, faintly disappointed
   - *sighs* and italicized asides
   - Formal but not stiff

2. **Complete Structure**
   - Opening statement
   - All personalities arguing
   - Cross-examination (when relevant)
   - Voting with proper format
   - Ruling with decision, rationale, risk

3. **Quality Arguments**
   - 3-5 lines per personality
   - Specific to the matter
   - Within personality's domain

4. **Realistic Interaction**
   - Personalities disagree
   - Cross-examination reveals gaps
   - Synthesis emerges naturally

## Contributing Examples

To add a new example:

1. **Conduct a real deliberation** — Examples should be genuine, not contrived
2. **Save the transcript** — Format properly
3. **Review for quality** — Does it demonstrate good practice?
4. **Add to this directory** — With appropriate naming
5. **Update this README** — Document what it demonstrates

### Naming Convention

`topic-deliberation.md` (lowercase, hyphens)

Examples:
- `database-selection-deliberation.md`
- `api-versioning-deliberation.md`
- `authentication-approach-deliberation.md`

## Reading Tips

### For New Users

Start with `feasibility-scale-deliberation.md`:
- It's comprehensive
- It shows all features
- It created a real framework (MFAF)

### For Specific Patterns

Look for examples that match your situation:
- Similar topic
- Similar complexity
- Similar controversy level

### For Voice Calibration

If your deliberations don't "sound right":
- Read examples aloud
- Note the rhythm and word choice
- Practice matching the tone
